# 即梦 Seedance 2.0 参考模式（全能参考）

根据 Seedance 2.0 手册提取的参考能力说明，用于生成 prompt。

---

## 📷 参考图像

参考图像可精准还原画面构图、角色细节。

**应用场景：**
- 使用实拍产品照片（raw/）作为质感参考
- 使用产品设计图（keyframes/）作为构图参考
- 参考图像应清晰展示产品细节、光影、材质

---

## 🎥 参考视频

参考视频支持镜头语言、复杂的动作节奏、创意特效的复刻。

**应用场景：**
- 可以使用参考视频来复制运镜和动作
- 支持镜头切换（推、拉、摇、旋转等）
- 支持动作复刻（手势、表情、运动轨迹）

---

## ⏱ 视频编辑能力

视频支持平滑延长与衔接，可按用户提示生成连续镜头，不止生成，还能"接着拍"。

**应用场景：**
- 可以指定特定时间段的动作和运镜
- 支持多段视频的无缝衔接
- 支持在现有视频基础上增加内容

---

## ✂️ 编辑能力

编辑能力同步增强，支持对已有视频进行角色更替、删减、增加。

**应用场景：**
- 可以修改视频中的特定角色
- 支持删除不需要的片段
- 支持增加新的镜头或特效

---

## 💡 使用建议

在 prompt 中引用参考模式时：

1. **明确使用参考图像**
   - 在 prompt 中通过 `(@image.jpg)` 引用参考图
   - 描述参考图的构图和细节要求

2. **描述想要的视频效果**
   - 指定镜头语言（如"推镜头"、"俯拍"）
   - 描述动作节奏（如"缓慢移动"、"快速切换"）
   - 描述创意特效（如"光影过渡"、"镜头光斑"）

3. **使用"全能参考"模式**
   - 在 modelConfig 中设置 `"referenceMode": "全能参考"`
   - 这个模式会综合所有参考素材进行生成

---

## 📝 示例 Prompt 片段

### 使用参考图像
```
(@raw_photo_001.jpg) (@raw_photo_002.jpg) (@raw_photo_003.jpg)
产品镜头展示。参考图展示了产品的不同角度和材质。使用相同的构图和布光，确保产品质感真实。
```

### 使用参考视频
```
(@reference_video.mp4)
运镜参考。从参考视频中提取第一段的推镜头动作，缓慢接近产品，最后停在产品特写。
```

### 组合使用
```
(@keyframes/render01.jpg) (@reference_video.mp4)
构图参考产品渲染图，运镜参考视频中前 10 秒的摇镜头动作。
```
